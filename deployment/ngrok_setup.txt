# Ngrok Setup Instructions

This file explains how to deploy the Streamlit app with Ngrok for public access.

---

## 1. Run the Streamlit App Locally

Navigate to the `Heart_Disease_Project` folder and start the app:

```bash
cd Heart_Disease_Project
streamlit run app.py
```

By default, the app will be running at:

    [http://localhost:8501](http://localhost:8501)

---

## 2. Install Ngrok

### Option A: Download Ngrok

* Visit [https://ngrok.com/download](https://ngrok.com/download)
* Download and install for your OS

### Option B: Install via pip

```bash
pip install pyngrok
```

---

## 3. Authenticate Ngrok (one-time setup)

Sign up at [https://ngrok.com](https://ngrok.com) to get your auth token. Then run:

```bash
ngrok config add-authtoken YOUR_AUTH_TOKEN
```

---
